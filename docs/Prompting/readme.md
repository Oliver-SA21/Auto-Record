# Uso de Inteligencia Artificial (Prompting)

## Descripción general

Durante el desarrollo del proyecto **Auto-Record**, se utilizó Inteligencia Artificial como herramienta de apoyo para la resolución de problemas técnicos, diseño de interfaz y estructuración de la documentación.

El uso de IA permitió optimizar el tiempo de desarrollo, mejorar la calidad del código y facilitar la toma de decisiones en distintas etapas del proyecto. Todas las respuestas generadas fueron analizadas, adaptadas y modificadas para integrarse correctamente en el sistema.

---

## Objetivo del uso de IA

El uso de IA en el proyecto tuvo como finalidad:

- Apoyar en la generación de ideas técnicas
- Resolver dudas sobre implementación
- Mejorar la estructura del código
- Optimizar el diseño de la interfaz
- Redactar documentación de forma clara y profesional

---

## Prompts utilizados

A continuación, se presentan algunos de los prompts más relevantes utilizados durante el desarrollo del sistema:

---

### 1. Diseño de API REST

**Prompt:**
Ayúdame a diseñar una API REST para un sistema de control de transporte que permita gestionar incidencias, unidades y registros usando Node.js y MySQL.

### 2. Implementación de CRUD en Express

**Prompt:**
Genera un ejemplo de CRUD en Node.js usando Express y MySQL para una tabla de incidencias.

### 3. Conexión a base de datos

**Prompt:**
Cómo conectar Node.js con MySQL usando mysql2 y async/await.

### 4. Diseño de interfaz con Tailwind

**Prompt:**
Dame ideas para diseñar una interfaz web moderna usando Tailwind CSS con tarjetas y colores profesionales.

### 5. Panel visual de API

**Prompt:**
Cómo crear una interfaz en HTML y JavaScript que permita consumir una API usando fetch y mostrar la respuesta en JSON.

---

## Conclusión

El uso de IA en el proyecto Auto-Record permitió acelerar el desarrollo, mejorar la calidad del sistema y facilitar la resolución de problemas técnicos. Su integración fue clave para lograr un producto funcional, organizado y bien documentado.
